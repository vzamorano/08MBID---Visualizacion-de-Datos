Justificación de los cambios en los gráficos

Gráfico 1: Pto. Distribuidor por Género

Los cambios realizados en este gráfico fueron mas visuales, es decir para que los colores, tamaños y estilos sean los adecuados para poder mejorar la visualización del gráfico de barras. Estos cambios fueron los siguientes:

  • Cambio de posición del título al centro del gráfico: Este cambio se realizó para tener una mejor apreciación del título, ya que es lo primero que observa un usuario, por eso también tiene un tamaño de fuente grande.
  • Aumento de tamaño de letra para la leyenda "Género" y cambio de posición a lado derecho: Con este cambio se puede apreciar de mejor manera la leyenda Género, ya que es necesario que estos datos sean fáciles de visualizar, por eso también se decidió moverlo al lado derecho del gráfico.
  • Aumento de tamaño de letra para los valores del eje "X" y "Y": En la clase se realizó el aumento de tamaño del eje X "Presupuesto", pero para que haya una consistencia en estos valores se realizó un ajuste en ambos ejes para que tengan el mismo tamaño de letra.
  • Aumento de tamaño de letra para los nombres del eje "X" y "Y": Se realizó el aumento de tamaño de los nombres de los valores de ambos ejes para que el usuario pueda visualizarlos de mejor manera.
  • Mostrar opciones de etiqueta de datos: Se decidió agregar los valores para cada sección de las barras, y asi de esta manera se puede identificar mejor la información de este gráfico.
  • Zoom slider para los valores del eje X: "Presupuesto": Agregar esta opción me pareció interesante porque gracias a este control deslizante, podemos observar a los distribuidores respecto a su presupuesto realizando un zoom específico por presupuesto.
  • Agregar sombra al borde del gráfico: Con esto se podrá distinguir de mejor manera de un gráfico a otro.
